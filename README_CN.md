## 第二部分：数据清洗

### 缺失值识别

原始数据存在一定比例的缺失值，需要首先了解具体情况：

```python
# 统计每列缺失值数量
df.isna().sum()
```

结果：

| 字段 | 缺失数量 | 缺失比例 |
|------|----------|----------|
| PM2.5 | 8,739 | 2.08% |
| PM10 | 6,449 | 1.53% |
| SO2 | 9,021 | 2.14% |
| NO2 | 12,116 | 2.88% |
| CO | 20,701 | 4.92% |
| O3 | 13,277 | 3.16% |
| TEMP | 398 | 0.09% |
| wd | 1,822 | 0.43% |
| 其他 | <400 | <0.1% |

可以看到，PM2.5作为核心污染物，缺失比例约为2%，数据完整性较好。CO和NO2的缺失相对较多，但仍在可接受范围内。

### 缺失值处理方法

数据清洗的具体步骤：

```python
# 步骤1：删除PM2.5为空的行（核心指标不能缺失）
df.dropna(subset=['PM2.5'], inplace=True)

# 步骤2：删除缺失率超过30%的行
df = df[df.isna().mean(axis=1) < 0.3]

# 步骤3：按时间排序
df.sort_values(by=['year','month','day','hour'], inplace=True)

# 步骤4：前向填充（用前一个值填充）
df.ffill(inplace=True)

# 步骤5：后向填充（用后一个值填充）
df.bfill(inplace=True)

# 步骤6：删除仍有缺失的行
df.dropna(inplace=True)
```

这种方法确保了时间序列的连续性，同时最大程度保留了有效数据。

## 第六部分：相关性分析

### 分析方法

```python
# 计算相关性矩阵
corr = df[['PM2.5','PM10','SO2','NO2','CO','O3','TEMP','PRES','DEWP']].corr()

# 查看PM2.5与各变量的相关性
corr['PM2.5'].sort_values(ascending=False)
```

### 相关性结果

| 排名 | 变量 | 相关系数 | 相关性强度 |
|------|------|----------|-----------|
| 1 | PM2.5 | 1.00 | 自身 |
| 2 | PM10 | 0.88 | 强正相关 |
| 3 | CO | 0.79 | 强正相关 |
| 4 | NO2 | 0.67 | 中等正相关 |
| 5 | SO2 | 0.48 | 弱正相关 |
| 6 | DEWP | 0.11 | 弱正相关 |
| 7 | PRES | 0.02 | 几乎无相关 |
| 8 | TEMP | -0.13 | 弱负相关 |
| 9 | O3 | -0.15 | 弱负相关 |

### 可视化方法

```python
# 绘制相关性热力图
plt.figure(figsize=(10,8))
sns.heatmap(corr, annot=True, cmap='coolwarm', center=0)
plt.title('Correlation Matrix')
plt.show()
```

### 关键发现

**1. PM10是PM2.5的"双胞胎"**

相关系数高达0.88，这意味着：
- PM10和PM2.5几乎同步变化
- 它们有共同的排放源（机动车尾气、扬尘、工业等）
- 控制PM10将有效降低PM2.5

**2. CO与PM2.5高度相关（0.79）**

一氧化碳同样来自不完全燃烧，可能与机动车尾气相关。

**3. 温度与PM2.5负相关（-0.13）**

这是重要发现：
- 天气越冷，PM2.5越高
- 解释了为什么冬季雾霾频繁
- 冬季供暖期是污染防治的关键时期

**4. O3与PM2.5负相关（-0.15）**

臭氧可能在光化学作用下分解颗粒物，具体机制需要进一步研究。

## 总结与结论

### 数据讲述的故事

1. **北京污染的空间格局**：市区污染显著高于郊区，核心区（东四）比郊区（定陵）高出近30%

2. **时间规律**：
   - 冬季污染高于夏季
   - 早晚高峰污染较高
   - 存在周期性波动

3. **主要矛盾**：PM2.5和PM10是主要污染物，它们的变化几乎同步

4. **防治启示**：
   - 控制PM10是降低PM2.5的关键
   - 冬季供暖期需要特别关注
   - 郊区可作为清洁空气的参考基准

### 数据局限性

- 缺少2018年后的数据
- 部分站点数据缺失较多
- 未包含污染源解析信息

### 后续建议

- 结合气象数据做深入分析
- 建立PM2.5预测模型
- 对比不同治理政策的效果
